Windows與AppServ環境下，關閉PHP過濾敏感字元之設定方式

1. 開始\所有程式\AppServ\Configuration Server\PHP Edit the php.ini Configuration File

3. 透過 Ctrl + F 找到「magic_quotes_gpc」，將其值設定為「Off」
